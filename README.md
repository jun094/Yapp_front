## 프론트 작업한거 확인하는 방법
    1. 패키지 설치 : npm install
    
    2. 실행 : npm run dev

    3. 인터넷 주소창에 localhost:3000 입력 

## 그 이외 유의사항은 각 폴더에 index.js파일에 주석으로 달아 놓았습니다
